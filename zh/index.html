# 介绍

ahooks 的 vue 实现。

许多 hooks 是从ahooks移植过来的，但是不完全一致。

# 特点

  • 易于使用.
  • 同时适用于 Vue3 和 Vue2。( 感谢vue-demi ).
  • 由 TS 编写，完整的类型推导.

# 安装

yarn add ahooks-vue

# 使用

import { useWorkerFunction } from 'ahooks-vue';
